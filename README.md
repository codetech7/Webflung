# Webflung

This repository is my first webflow repository

## Project Overview

Webflung is a basic Webflow-inspired project repository designed to demonstrate modern web development practices. This project serves as a foundation for building responsive, interactive web experiences.

## Structure

- `index.html` - Main entry point with semantic HTML structure
- `main-design.css` - Custom stylesheet with CSS variables and responsive design
- `interaction-handler.js` - JavaScript for interactive features and smooth animations

## Features

- Responsive navigation menu
- Smooth scroll behavior
- Interactive project cards with hover effects
- Scroll-reveal animations
- Mobile-friendly responsive design
- Modern CSS with custom properties

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Explore the interactive features and responsive design

## Customization

You can customize the design by modifying the CSS variables in `main-design.css`:

```css
:root {
    --primary-tone: #2d5a8c;
    --secondary-tone: #4a90d9;
    --accent-highlight: #e8f1f8;
}
```

## Browser Compatibility

This project works best in modern browsers that support:
- CSS Grid and Flexbox
- CSS Custom Properties
- Intersection Observer API
- ES6 JavaScript features
